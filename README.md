# 🌌 Acerca de RelatosHub

**RelatosHub** es una plataforma digital enfocada en la publicación y lectura de relatos, cuentos cortos, experiencias personales, historias de ficción y narrativas originales.

Su objetivo es conectar escritores y lectores en un mismo espacio, permitiendo compartir historias, interactuar con otros usuarios y construir una biblioteca comunitaria de relatos.

La plataforma permite:

- 📖 Leer relatos de diferentes categorías
- ✍️ Publicar historias propias
- ❤️ Dar me gusta a publicaciones
- 💬 Comentar relatos
- 🔖 Guardar favoritos
- 👤 Seguir autores
- 🏷️ Explorar categorías y etiquetas
- 🔍 Buscar contenido fácilmente

# ⚡ Instalación

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/Pagina-Relatos
```

---

## 2️⃣ Instalar dependencias

```bash
npm install
```

---

## 3️⃣ Configurar variables de entorno

```env
VITE_API_URL=http://localhost:3000
```

---

## 4️⃣ Ejecutar aplicación

```bash
npm run dev
```

---

## 5️⃣ Abrir navegador

```bash
http://localhost:5173
```
